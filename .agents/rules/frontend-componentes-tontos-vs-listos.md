En Vue 3, divide y vencerás. Ten componentes "tontos" (UI) que solo reciben props y emiten events (ej: un botón, una tarjeta de pedido), y componentes "listos" (Vistas/Páginas) que se conectan a la API o al store. Así puedes reutilizar los botones sin arrastrar lógica de negocio.
