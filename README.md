# Komanda
KOMANDA 🚀 | SaaS modular para gestión de restaurantes. Control de inventarios con Back-flushing automático por receta, POS integrado, gestión de nómina por comisiones y sistema de comandas en tiempo real. Stack: Vue.js (TS), PHP/Typescript &amp; PostgreSQL. Arquitectura limpia, escalable y pensada para la eficiencia operativa. 🍽️⚡️
